## Instalasi

1. Clone repositori
2. Jalankan `npm install` untuk menginstal semua dependensi yang diperlukan.

## Menjalankan Tes

Menjalankan tes, gunakan perintah berikut:

```sh
npx cypress open
```

atau dengan

```sh
npx cypress run
```
